FACILITIES / GALLERIES / SAND POINT GALLERY

Thumbnail + Lightbox Gallery Demo

This is a demonstartion of a standard thumbnail gallery using colorbox. This is one of many image display options we offer by default in our service. Images can be uploaded, arranged, given captions, and cropped through the editor interface. It is visually customizable, responsive, and built with mobile views in mind.

With this option a button could link the main Smugmug page. This can operate as a point of entry to your larger body of images hosted off-site.
